# Zomato Restaurant Data Analysis (EDA)

# Project Overview : 
This project focuses on cleaning, analyzing, and visualizing the Zomato Restaurant Dataset to uncover meaningful business insights about restaurant performance, customer preferences, pricing patterns, and location-based trends.

The analysis was conducted using Python and popular data analytics libraries such as Pandas, NumPy, Matplotlib, and Seaborn.

# Dataset : 
The dataset used in this project is sourced from Kaggle.
